# Nexus Hotel UI

A modern React-based hotel booking application with responsive design and Stripe payment integration.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Room Management**: Browse and book hotel rooms with detailed descriptions
- **Admin Dashboard**: Manage rooms, bookings, and user profiles
- **Payment Integration**: Secure Stripe payment processing
- **User Authentication**: Login/registration system with profile management
- **Booking Management**: Search and manage bookings

## Technologies Used

- **Frontend**: React 19, CSS3, Font Awesome
- **Payment**: Stripe.js integration
- **Routing**: React Router DOM
- **HTTP Client**: Axios
- **Build Tool**: Create React App

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Aghniv/nexus-ui.git
cd nexus-ui
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

## Deployment

To build for production:
```bash
npm run build
```

The build folder will contain the optimized production files.
